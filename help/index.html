Say hi to Gmelius for Firefox.

Configuration

To configure Gmelius in Firefox, go to Tools Add-ons Extensions Gmelius and click on "preferences". You will need to reload Gmail after changing the preferences.

Privacy

To be clear and to the point, Gmelius will never access, read, store, alter or transmit your personal data. If you are a Chrome user, the "Your tabs and browsing activity" permission comes from the fact that this extension modifies the style and user interface of Gmail within your browser and so needs to know whether Gmail is open in one or several of your tabs to apply those changes.

Moreover, Gmelius code meets the latest Content Security Policy (CSP) recommendations, making sure its users can keep browsing the Web safely.

Support

This is the place to leave your feedback, suggestions and bugs. Before posting anything, please ensure to download and run the latest version of Gmelius. Always include the operating system, browser version and Gmelius version you are using when sharing a bug, e.g. Windows 8 - Firefox 22 - Gmelius 5.7.5.
